# Fullscreen Chatbot WordPress Plugin

A minimalistic and stylish fullscreen chatbot interface powered by OpenAI for your WordPress site.

## Features

- **Fullscreen chatbot interface** - Clean, modern design that takes over the entire page
- **OpenAI integration** - Supports all major OpenAI models (GPT-4o, GPT-4, GPT-3.5, etc.)
- **Easy configuration** - Simple admin dashboard to manage settings
- **Page selector** - Choose which page displays the chatbot via dropdown
- **Customizable system prompt** - Define your AI's personality and behavior
- **Conversation history** - Maintains context during the chat session
- **Responsive design** - Works perfectly on desktop, tablet, and mobile devices

## Installation

1. Download the `fullscreen-chatbot.zip` file
2. Log in to your WordPress admin panel
3. Go to **Plugins > Add New**
4. Click **Upload Plugin** at the top of the page
5. Choose the `fullscreen-chatbot.zip` file
6. Click **Install Now**
7. After installation, click **Activate Plugin**

## Configuration

1. In your WordPress admin panel, click **Chatbot** in the left sidebar
2. Enter your **OpenAI API Key** (get one at https://platform.openai.com/api-keys)
3. Select your preferred **OpenAI Model** (GPT-4o recommended)
4. Choose which **page** should display the chatbot from the dropdown
5. (Optional) Customize the **System Prompt** to define your chatbot's behavior
6. Click **Save Changes**

## Usage

Once configured, navigate to the page you selected in the settings. The page will automatically display the fullscreen chatbot interface instead of its normal content.

Users can:
- Type messages and receive AI-powered responses
- View conversation history during their session

- WordPress 5.0 or higher
- PHP 7.4 or higher
